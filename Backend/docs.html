API Documentation

Welcome to the API Documentation

Endpoints

Get All Blog Posts

URL: /api/posts

Method: GET

Success Response:

[
  {
    "_id": "123",
    "title": "Post Title",
    "content": "Post Content",
    "author": "Author Name",
    "image": "Image URL",
    "likes": [],
    "comments": []
  }
]

Register a New User

URL: /api/register

Method: POST

Request Body:

{
  "username": "User",
  "email": "user@example.com",
  "password": "password"
}

Success Response:

{
  "_id": "123",
  "username": "User",
  "email": "user@example.com",
  "password": "hashed_password",
  "__v": 0
}

Error Response: Username or Email is already registered

Login

URL: /api/login

Method: POST

Request Body:

{
  "email": "user@example.com",
  "password": "password"
}

Success Response:

{
  "_id": "123",
  "username": "User",
  "email": "user@example.com",
  "password": "hashed_password",
  "__v": 0
}

Error Response: Invalid email or password

Create Blog Post

URL: /api/posts

Method: POST

Request Body:

{
  "title": "Post Title",
  "content": "Post Content",
  "author": "Author Name",
  "image": "Image URL"
}

Success Response:

{
  "_id": "123",
  "title": "Post Title",
  "content": "Post Content",
  "author": "Author Name",
  "image": "Image URL",
  "likes": [],
  "comments": [],
  "__v": 0
}

Error Response: Bad Request

Like a Blog Post

URL: /api/posts/:postId/like

Method: POST

URL Params: postId=[String]

Success Response:

{
  "_id": "123",
  "title": "Post Title",
  "content": "Post Content",
  "author": "Author Name",
  "image": "Image URL",
  "likes": [{"user": "SomeUser"}],
  "comments": [],
  "__v": 0
}

Error Response: Blog post not found or Internal server error

Dislike a Blog Post

URL: /api/posts/:postId/dislike

Method: POST

URL Params: postId=[String]

Success Response:

{
  "_id": "123",
  "title": "Post Title",
  "content": "Post Content",
  "author": "Author Name",
  "image": "Image URL",
  "likes": [],
  "comments": [],
  "__v": 0
}

Error Response: Blog post not found or Internal server error

Add Comment to Blog Post

URL: /api/posts/:postId/comments

Method: POST

URL Params: postId=[String]

Request Body:

{
  "content": "Comment content",
  "author": "Comment Author"
}

Success Response:

{
  "_id": "123",
  "title": "Post Title",
  "content": "Post Content",
  "author": "Author Name",
  "image": "Image URL",
  "likes": [],
  "comments": [
    {
      "_id": "456",
      "content": "Comment content",
      "author": "Comment Author",
      "date": "2024-05-27T14:48:00.000Z"
    }
  ],
  "__v": 0
}

Error Response: Blog post not found or Internal server error

Delete Comment from Blog Post

URL: /api/posts/:postId/comments/:commentId

Method: DELETE

URL Params: postId=[String], commentId=[String]

Success Response:

{
  "_id": "123",
  "title": "Post Title",
  "content": "Post Content",
  "author": "Author Name",
  "image": "Image URL",
  "likes": [],
  "comments": [],
  "__v": 0
}

Error Response: Blog post not found, Comment not found, or Internal server error

Submit Contact Form

URL: /submit

Method: POST

Request Body:

{
  "name": "Your Name",
  "email": "your.email@example.com",
  "message": "Your message"
}

Success Response:

{
  "message": "Data saved successfully"
}

Error Response: Internal Server Error
